# AlterBake Work Profile

Eksperymentalny MVP narzędzia do tworzenia profilu pracy / kwestionariusza (kategorie: niezawodność, odporność na presję itd.). Aplikacja statyczna HTML/CSS/JS bez backendu. `index.html` to wersja wieloplikowa (korzysta z `app.js`, `questions.js`, `style.css`); `profil-pracy-mvp.html` to samowystarczalny wariant jednoplikowy.

Status: wczesny MVP / eksperyment — kod przeniesiony z repo `alterbake-menu-board`, gdzie powstał przez pomyłkę.
